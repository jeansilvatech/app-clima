# App Clima

### Aplicação web para consulta de previsão do tempo pesquisando por cidade, retornando: 

* Nome da cidade
* Bandeira do país
* Temperatura
* Umidade
* Velocidade do vento
* Fundo de acordo com o tempo

### Tecnologias utilizadas:
* HTML
* CSS
* JavaScript
* Weather API
